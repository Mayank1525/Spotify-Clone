# 🎧 Spotify Clone — Frontend Music Streaming Interface

## 📌 Overview

The **Spotify Clone** is a frontend-centric music streaming web application developed using **vanilla HTML, CSS, and JavaScript**. It replicates the core user interface and interactive behavior of modern music platforms, with a focus on **clean architecture, responsive design, and performant client-side rendering**.

This project demonstrates strong proficiency in **DOM manipulation, event-driven programming, and UI/UX engineering without relying on frameworks**.

---

## 🚀 Key Features

* 🔍 **Dynamic UI Rendering** — Interactive song selection and navigation
* 🎵 **Custom Audio Player** — Play, pause, seek, skip, and volume control
* 📂 **Playlist Simulation** — Organized display of tracks and albums
* 📱 **Responsive Design** — Optimized for mobile, tablet, and desktop
* ⚡ **Lightweight & Fast** — No frameworks, minimal dependencies
* 🎨 **Modern UI/UX** — Spotify-inspired layout with smooth transitions

---

## 🏗️ Tech Stack

* **HTML5** — Semantic structure and accessibility
* **CSS3** — Flexbox, Grid, animations, and responsive design
* **JavaScript (ES6+)** — Core logic, DOM manipulation, and event handling

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/spotify-clone.git
cd spotify-clone
```

### 2️⃣ Run the Project

Simply open the `index.html` file in your browser:

```bash
start index.html
```

*or*

Double-click `index.html`
